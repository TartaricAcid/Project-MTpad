#Instructions#
* 按下空格以使用补全，但只在你至少键入一个字母时有效

* HighLights存储着所有关键字和它们的颜色，格式如下<br>
``` VisualBasic
   ->  
   R G B  
   关键字1  
   关键字2  
   关键字3  
   ...  
```

* Statements存储着所有的关键字的从属关系，格式如下<br>
``` VisualBasic
    关键字1  
    ->  
    属于关键字1的子关键字1  
    ->  
    属于上个子关键字的关键字  
    <-  
    属于关键字1的子关键字2  
    <-  
    关键字2  
    ...  
```
